## Problem
Given two integers `a` and `b`, return _the sum of the two integers without using the operators_ `+` _and_ `-`.

**Example 1:**
**Input:** a = 1, b = 2
**Output:** 3

**Example 2:**
**Input:** a = 2, b = 3
**Output:** 5

**Constraints:**
- `-1000 <= a, b <= 1000`

## Solution
We are gonna use the `XOR` operator and `AND` operator to sum the numbers:

```java
class Solution {
    public int getSum(int a, int b) {
        while (b != 0) {
            int tmp = (a & b) << 1;
            a = a ^ b;
            b = tmp;
        }

        return a;
    }
}
```
