## Problem
You are given an integer array `nums` where the largest integer is **unique**.

Determine whether the largest element in the array is **at least twice** as much as every other number in the array. If it is, return _the **index** of the largest element, or return_ `-1` _otherwise_.

**Example 1:**
**Input:** nums = [3,6,1,0]
**Output:** 1
**Explanation:** 6 is the largest integer.
For every other number in the array x, 6 is at least twice as big as x.
The index of value 6 is 1, so we return 1.

**Example 2:**
**Input:** nums = [1,2,3,4]
**Output:** -1
**Explanation:** 4 is less than twice the value of 3, so we return -1.

**Constraints:**
- `2 <= nums.length <= 50`
- `0 <= nums[i] <= 100`
- The largest element in `nums` is unique.

## First Solution
We can keep track of the two biggest values in the array and the index of the biggest one. Then we just need to check if the biggest one is 2 times larger than the second one:

```python
class Solution:
    def dominantIndex(self, nums: List[int]) -> int:
        if nums[0] > nums[1]:
            first, second = nums[0], nums[1]
            index = 0
        else:
            first, second = nums[1], nums[0]
            index = 1

        
        for i in range(2, len(nums)):
            if nums[i] > first:
                second, first = first, nums[i]
                index = i
            elif nums[i] > second:
                second = nums[i]
        
        return index if first >= second * 2 else -1
```

## Second solution
We can make use of the python libraries, to first make a copy `aux` of the initial array `nums`, that we will later sort. Then we will return the index, of the largest element of `aux` if its 2 times bigger than the next element,  in the array `nums`, `-1` if not:

```python
class Solution:
    def dominantIndex(self, nums: List[int]) -> int:
        aux = nums.copy()
        aux.sort(reverse=True)

        return nums.index(aux[0]) if aux[0] >= aux[1]*2 else -1
```
